### General
i : insert mode left
a : insert mode right
v : visual mode
V : visual mode(block)

### Normal Mode

y : yank (copy)
p : paste

## NVIM KEYMAPS
### Default
space + f5 => source init.lua
space + esc => stop find & replace

### Telescope
space + ff => find_files
space + fg => live_grep

### Neotree
space + ee => open file explorer

### LSP
ctrl + k => hover information
space + df => code definitions
space + ca => code actions
